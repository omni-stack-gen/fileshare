# Q5kNvw 布局分析报告

目标节点：`Q5kNvw`
名称：`IP Num - Input`
用途：CCTV 设置的 IP 地址数字输入弹窗内部容器。

## Visual Hierarchy

- `Q5kNvw` frame, 380 x 486, fill `#121c2cff`, radius 24, stroke `#ffffff1f`, clip true
  - `RP2do` 顶部容器, 378 x 133, padding `[24,28,0,28]`, vertical gap 16
    - `uXM27` 标题容器
      - `gI85o` text `IP 주소 입력`, `#ffffff73`, Inter 14/500
    - `LVtqW` IP 四段输入容器, 322 x 52, horizontal, centered, gap 6
      - `s8Yq8` 第一段 + 点，段框 `vqLx2` 68 x 52，内容 `192`
      - `htMVf` 第二段 + 点，段框 `bE98d` 68 x 52，内容 `168`
      - `WNuXY` 第三段 + 点，段框 `BU44h` 68 x 52，内容 `1`
      - `E0u9kY` 第四段，68 x 52，内容 `100`，当前输入段，fill `#51a2ff1f`，stroke `#51a2ff99` 1.5
  - `Lhe98` 分割线, 378 x 1, fill `#ffffff12`
  - `kfp7h` 数字键盘容器, 378 x 280
    - 1-9、`.`、0、退格键，按钮 106 x 52，列 x=20/136/252，行 y=20/82/144/206
  - `Nl9w4` 底部确认容器, 378 x 72
    - `XIPNz` 确认按钮, x=20, width=338, height=52, fill `#155dfcff`, radius 12

## 坐标与属性重点

- 弹窗内部四段 IP 区域使用 4 个独立输入框，单段宽 68，高 52。
- 段与段之间点号为独立文本，颜色 `#ffffff4d`，Inter 20/600。
- 未激活段：fill `#ffffff0d`，stroke `#ffffff1a` 1。
- 激活段：fill `#51a2ff1f`，stroke `#51a2ff99` 1.5。
- `pJGE2` 的空输入状态显示第一段为 `|` 且激活，其余三段显示 `0`，确认按钮禁用。

## 实现要求

- Slint 中 IP 弹窗不得渲染为单个字符串输入框。
- 输入状态需要保存 4 个 IP 段和当前激活段索引。
- 数字输入只追加到当前段，`.` 切换到下一段，退格删除当前段内容。
