## Popup Nodes

- `pJGE2`: IP keypad overlay, dim backdrop, centered numpad `380 x 486`, decimal key enabled.

## Interaction Logic From work.md

`MpUw5` navigates to `UGcOs`. `Ia1RF`, `ddmM7`, `aSKvN`, `EkZIL` open their respective input overlays. Account/password keyboards preserve WiFi-style lower/upper/symbol switching. When all four values exist, `O4p9d` switches to `hohrT`; clicking it shows `E0zNY` or `hMGBM`. Saving from success state persists settings and shows `foYV2` saved state.
